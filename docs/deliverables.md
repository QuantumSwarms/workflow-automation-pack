# Deliverables — Back-Office Automation for SMBs

Each item below is an acceptance gate. Engagement is complete when all are checked and the handoff doc is signed.

- [ ] Process map (before / after) as a single PDF.
- [ ] Live automations in your tooling, version-controlled where possible.
- [ ] Runbook explaining each automation in plain English.
- [ ] Hours-saved tracker with 30-day measurement.
- [ ] Loom video walkthrough for your team.

## Handoff doc

A short PDF you sign at the end confirming:
- Every deliverable above is delivered and accepted.
- 30-day warranty period start date.
- Final invoice (if balance remains) is due Net-15.
- Optional: testimonial / case study consent (you're never required to give one).
