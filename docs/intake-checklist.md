# Intake Checklist — Back-Office Automation for SMBs

Bring these to the kickoff call. Anything missing extends the timeline.

- [ ] List of tools currently in use (CRM, invoicing, scheduling, email, storage).
- [ ] Admin access or admin contact for each tool.
- [ ] One day where we can shadow operations (in person or via screen-share).
- [ ] Owner availability for kickoff + final review.

## Logistics

- [ ] Mutual NDA signed (our template or yours).
- [ ] Statement of Work signed and 50% deposit cleared.
- [ ] Primary contact + escalation contact identified (name, email, phone).
- [ ] Slack / Teams / email — pick one channel for the engagement.
- [ ] Shared folder created (Drive / Notion / Dropbox).

## Kickoff agenda (90 min)

1. Introductions and roles (10 min).
2. Walk through this checklist together (15 min).
3. Confirm scope and acceptance criteria (20 min).
4. Working session on the highest-priority item (30 min).
5. Schedule recurring checks and define done (15 min).
