# Security Policy

## Reporting a vulnerability

Email **john@epochcoreqcs.com** with subject `[SECURITY] <repo-name>`.

- Do **not** open a public GitHub issue for security reports.
- Expect an initial response within **2 business days**.
- Please include reproduction steps, impact, and any suggested mitigation.

## Scope

This repository contains marketing and specification artifacts only. No production source code or live credentials are stored here. If you believe credentials or secrets have been committed, contact us immediately.
