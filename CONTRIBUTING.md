# Contributing

This repository is a **proprietary client-facing artifact** of EpochCore LLC. External code contributions are not accepted.

## For internal collaborators / subcontractors

1. Branch from `main` using `feat/<short-name>`, `fix/<short-name>`, or `docs/<short-name>`.
2. Use [Conventional Commits](https://www.conventionalcommits.org/) (e.g., `feat: add intake form`, `docs: update pricing tier`).
3. Open a PR against `main`. CI must pass. CODEOWNERS review required.
4. Squash-merge.

## For prospective clients

If you found this repo and want to engage:
- Open an issue using the **Engagement Inquiry** template, or
- Email **john@epochcoreqcs.com** with subject `[INQUIRY] <your-company>`.

## Confidentiality

Do not commit client names, credentials, internal pricing notes, or NDA-covered material to any branch — public or private.
